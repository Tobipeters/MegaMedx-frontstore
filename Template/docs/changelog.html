  • UpdatedSecurity patch: Added rel='noopener' to all links with target='_blank' attribute. For more info read this article
  • UpdatedNode modules to latest versions
  • FixedHandheld Toolbar styles: Removed padding-bottom that equals to Toolbar height from body by default. Now you can add bottom padding to body by applyoing css class toolbar-enabled to body. If you use PUG it applies automatically if var isToolbar = true is added to particular page template or globally enabled via templates/helpers/config.pug. It's done to ensure that you can remove bottom padding from body without editing core theme styles if you don't need Handheld Toolbar.
  • FixedSCSS maps in theme _variable.scss. Special thanks to Kevin Ramharak for the fix suggestion
  • FixedMinor bug fixes
